Что выведет программа? Объяснить вывод программы.

```go
package main

import (
    "fmt"
)

func main() {
    a := [5]int{76, 77, 78, 79, 80}
    var b []int = a[1:4]
    fmt.Println(b)
}
```

Ответ:
```
Вывод:
[77 78 79]
Запись a[1:4] соответсвует полуинтервалу [1, 4)
т.е. берутся элементы с номерами 1,2,3

```
