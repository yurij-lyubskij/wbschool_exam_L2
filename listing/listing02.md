Что выведет программа? Объяснить вывод программы. Объяснить как работают defer’ы и их порядок вызовов.

```go
package main

import (
	"fmt"
)


func test() (x int) {
	defer func() {
		x++
	}()
	x = 1
	return
}


func anotherTest() int {
	var x int
	defer func() {
		x++
	}()
	x = 1
	return x
}


func main() {
	fmt.Println(test())
	fmt.Println(anotherTest())
}
```

Ответ:
```
Вывод: 
2
1
Выражение defer добавляет вызов функции после ключевого слова defer в стек отложенных вызовов.
Все функции в стеке вызываются при возврате из функции, в которой они добавлены. 
Поскольку вызовы помещаются в стек, они производятся в порядке от последнего к первому.
В первом случае работаем с именованными возвращаемыми параметрами, поэтому здесь x - это сам результат, 
и его изменение в defer закономерно влияет на то, что вернет функция.
Во втором случае мы изменяем локальную переменную x уже после того, 
как return отработал и взял ее копию, которую и вернет, поэтому вызов defer не влияет на возвращаемое значение.

```
